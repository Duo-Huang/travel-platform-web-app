# 企业差旅服务平台Web

## 技术栈：

- 语言：TS + JS(fake server)

- 框架：vue3 + vitest

- 工程化：pnpm + vite

- css预处理：scss

## 环境

- Nodejs: v16+ (nvm recommend)

- Pnpm: latest(recommend)



## 本地开发：

1. 安装依赖

   ```bash
   pnpm i 
   cd fake-server
   pnpm i
   ```

   

2. 启动开发服务器 & fake server

   ```bash
   pnpm dev
   cd fake-server
   pnpm start
   ```

## 测试：

```bash
pnpm test
```
